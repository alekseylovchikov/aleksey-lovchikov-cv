## Generate PDF

Generate a PDF version of your CV using Playwright:

```bash
# Generate PDF from built version (requires npm run build first)
npm run pdf

# Generate PDF from dev server (requires npm run dev in another terminal)
npm run pdf:dev
```

The PDF will be saved as `cv-aleksey-lovchikov.pdf` in the project root.

**Note:** The first time you run the PDF generation, Playwright will download Chromium browser (~150MB).

## GitHub Pages

The project is automatically built and deployed to GitHub Pages when pushing to the `main` branch.

### GitHub Pages Setup

1. Go to Settings → Pages in your repository
2. In the "Source" section, select "GitHub Actions"
3. The workflow will automatically determine the correct base path:
   - If the repository is named `username.github.io`, base path will be `/`
   - Otherwise, base path will be `/repository-name/`

### Manual Deployment

You can manually trigger deployment via Actions → Build and Deploy to GitHub Pages → Run workflow.
